# Yew Starter Template
A base template to quickly start new web projects with **Rust + Yew**.

## Requirements
- Rust
- Trunk

**Trunk Setup**
```bash
cargo install trunk
rustup target add wasm32-unknown-unknown
```

**Local Execution**
```bash
trunk serve --open
```

**Production build**
```bash
trunk build --release
```
