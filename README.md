# Yew Starter Template
A base template to quickly start new web projects with **Rust + Yew**.

## Requirements
- Rust
- Trunk

### Install Trunk:
```bash
cargo install trunk
```

### Local Execution
```bash
trunk serve --open
```

### Production build
```bash
trunk build --release
```
